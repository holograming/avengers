## Avergers 
이 프로젝트는 멀티 에이전트 개발 시스템입니다.
Claude Code와 통합되어 복잡한 개발 작업을 자동화합니다.

---------------------------------------------------------------
## 참고 문서
- 세부 동작들을 구현하기 위해서 reference/ 폴더의 문서를 참고한다. 
  심도있게 docs 폴더를 읽고서, 전체적으로 최신 클로드 개발팀의 방법론을 적극적으로 도입한다. 
- token 절약을 위한 refenerece 문서에서 필요 부분들을 찾아서 tool + mcp + skills 등 조합을 활용하여
  토큰 절약하여 효율적인 시스템이 필요하다. 
- 업무의 세분화 및 병렬화된 작업을 진행시에, 워크트리를 이용한 병렬 작업 방식이 존재하기 때문에, 적극 도입한다.
  작업이 완료되면 기존 워크트리는 삭제하고, 하나로 머지하도록 한다.
