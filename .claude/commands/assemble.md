# Avengers Assemble

팀을 소집하고 임무 준비 상태를 확인합니다.

## 실행 내용

1. 모든 에이전트의 가용성 확인
2. 현재 진행 중인 태스크 요약
3. 팀 준비 상태 보고

## 에이전트 브리핑

각 에이전트에게 역할을 상기시키고 준비 상태를 확인합니다:

- **Captain**: 오케스트레이션 준비 완료?
- **IronMan**: 개발 환경 준비 완료?
- **Natasha**: 백엔드 시스템 접근 가능?
- **Groot**: 테스트 프레임워크 준비 완료?
- **Jarvis**: 정보 수집 도구 활성화?
- **Dr.Strange**: 전략 분석 준비 완료?
- **Vision**: 문서화 도구 준비 완료?

## 사용법

```
/assemble
```

팀이 소집되면 Captain이 브리핑을 시작합니다.
